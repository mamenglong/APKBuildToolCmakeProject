# APK Build Tools Cmake Example

```shell

# create build dir
mkdir build
cd build

# make app-unsigned.apk
make app-unsigned

# make example_dex.dex
make example_dex

# make app
make app

# debug apk (`android:debuggable="true"` must be settled)
make debug_apk_app

# install apk
make install_apk_app

```
